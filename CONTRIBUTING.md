# Contributing

Thank you for your interest in contributing to CastrOS Theme.

## Reporting Issues

If you find a bug, please open an Issue describing:

- Plasma version
- Linux distribution
- Steps to reproduce
- Screenshots (if applicable)

## Pull Requests

Before opening a Pull Request:

- Follow the existing project structure.
- Keep the visual style consistent.
- Test your changes before submitting.

## Coding Style

- Use descriptive commit messages.
- Keep SVG files clean and optimized.
- Do not remove original license information where required.

Thank you for helping improve CastrOS Theme.
